# offline-learn

Offline learning platform CLI tool

## Installation

```
bash
npm install
```

## Usage

```
bash
node bin/index.js
```

## Commands

- fetch - Download learning content
- search - Search for content
- test - Run tests on content
- list - List available content
- progress - Track learning progress
- export - Export learning content
- update - Update learning content
- publish - Publish learning content
